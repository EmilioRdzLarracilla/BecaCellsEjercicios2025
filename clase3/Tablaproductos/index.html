EJERCICIO TABLA COMPLEJA DE PRODUCTOS

Queremos crear una tabla que represente una lista de productos de una tienda. La tabla tendrá las siguientes características:

  • Encabezados que describan las columnas: Producto, precio, descripción y disponibilidad
  • Algunas celdas que estén combinadas usando colspan o rowspan
  • Diferentes estilos para algunas celdas (colores, bordes, etc.)
